# RaspberryPiMinecraft
Configuration files and performance testing scripts to setup and optimize a dedicated Minecraft (PC Java version) server

## To start server
`./start.sh`

## Console of minecraft
`screen -r minecraft`

to exit:
`ctrl+a d`
